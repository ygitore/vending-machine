# Vending-Machine
## Description
created application that allows users login using google authentication. Once user is logged in they can see the details of snacks name, location, price and image in vending machine
### Technologies used
* Html5, CSS3, Javascript
* Firebase 7.2.3
* Bootstrap4
* JQuery 3.4.1
